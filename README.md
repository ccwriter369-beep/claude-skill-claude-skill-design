# claude-skill-design

Claude Code skill for designing discoverable, well-structured Claude Code skills — embedding trigger phrases, structuring instructions, and maximizing skill activation reliability.

## Install

```bash
cp -r . ~/.claude/skills/claude-skill-design/
```

## Trigger Phrases

- "improve skill triggering"
- "fix skill discovery"
- "optimize my skill description"
- "make my skill trigger more often"
- "design a new skill"

## What's Included

- `SKILL.md` — guidelines for skill structure, trigger design, and instruction patterns
